Project Description
This repository contains the source code for my interactive personal portfolio. The website is designed to provide a comprehensive view of my academic background, technical skills, and professional leadership roles. It uses a modern dark-themed interface with 3D background effects and responsive design.

Key Sections
Hero Section: A professional introduction with an interactive background.

Intro Slider: A scrolling marquee highlighting my background and major brand affiliations.

Experience and Leadership: A detailed list of my roles with organizations like Programming Hero, Phitron, and oraimo.

Projects: A grid displaying my technical work and GitHub repositories.

Education: My academic history including SSC, HSC, and B.Sc. details.
